Course Description

This course enables students to design, implement, and analyze solutions to problems, use and implement commonly used algorithms, use standard data structures, develop and select appropriate algorithms and data structures to solve new problems, write solutions fluently in an object-oriented paradigm, write, run, test, and debug solutions in the Java programming language, utilizing standard Java library classes and interfaces from the AP Java subset, read and understand programs consisting of several classes and interacting objects, read, and understand a description of the design and development process leading to such a program, understand the ethical and social implications of computer use.

Prerequisites: Course taken in Introduction to Programming (either Java, Python, or C++), Skills including logic thinking, reasoning abilities, mathematical aptitude are assets in this course.

Preferred Programming Language: The course is run in Java, so that is preferred. If you are able to program in other programming languages, you are welcome to join this course.

Note: Please contact your instructor if you have not completed the prerequisites for this course.

Exam Date: May 15th, 2026 (note: you do not need to write the exam to take this course!)

Scoring

Scoring
Homeworks are marked as a percentage score; multiple choice part is combined with the coding/free response part. This is similar in tests. For exams, the grades are separated into Multiple Choice and Free Response, and they are weighed according to the College Board's criterion - you are then assigned a grade of 5, 4, 3, 2, or 1 (similar to the College Board assessment). Labs are graded with Pass or Incomplete.

Here's what they mean:

  • 5 - Extremely Well Qualified
  • 4 - Well Qualified
  • 3 - Qualified
  • 2 - Possibly Qualified
  • 1 - No Recommendation
Formula: ROUND(MC_Score * 1.0476 + FR_Score * 1.44)
