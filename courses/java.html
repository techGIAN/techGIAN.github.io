Course Description

This course is designed as an introductory programming course specifically in the programming language Java. In this course, students will learn basic programming concepts, such as variables, operators, conditional statements, writing iterative programs, simple data structures (e.g., arrays, lists, sets), recursion, basic object-oriented programming, and software testing. By the end of the course, students will gain programming skills needed in various courses such as AP-Computer Science, Computing Contest Courses, and ICS 3U.
