Course Description

This course is designed as an introductory programming course specifically in the programming language Java. In this course, students will learn basic programming concepts, such as variables, operators, conditional statements, writing iterative programs, simple data structures (e.g., arrays, lists, sets), recursion, basic object-oriented programming, and software testing. By the end of the course, students will gain programming skills needed in various courses such as AP-Computer Science, Computing Contest Courses, and ICS 3U.

Course Policies

  1. Please come on time (~10-15 minutes before the start of the class time).
  2. Have your laptop computer ready fully-charged (or at least with battery); if your laptop is close to 0%, make sure to bring a charger to class (always bring a charger!). Note that you a tablet or a phone cannot be used as a replacement for a laptop.
  3. Load Java/Eclipse IDE before the class begins (see Other Resources section below for more information about software installation).
  4. Phones (and other electronic devices aside from laptops) away during class! If you need to use them, ask your instructor for permission.
  5. The class is meant to be interactive. Participate in class discussions, and make sure to ask questions when you are not certain about something.
  6. Feel free to approach and ask your instructor questions before the class starts, during the break(s), and after class. You can also contact your instructor anytime through email (see his contact at the header of this course webpage)
  7. There will be coding exercises throughout the lectures. You are expected to try working on these exercises (as I do not necessarily give out the solutions) in-class. I will usually walk around to check on everybody.
  8. There are homeworks expected to be submitted weekly. They will be released every Tuesdays, and will be due the following Tuesday. More information about Homework submission in the Homeworks section below.
  9. There will be a midterm test and a final exam for this course. More information will be provided closer to the dates.
