Course Description

This course is designed as an introductory programming course specifically in the programming language Java. In this course, students will learn basic programming concepts, such as variables, operators, conditional statements, writing iterative programs, simple data structures (e.g., arrays, lists, sets, maps), recursion, basic object-oriented programming, and software testing. By the end of the course, students will gain programming skills needed in various courses such as AP-Computer Science and Computing Contest Courses. No prior programming experience needed.

Instructions for submission:

  1. First create a folder with the following filename: firstname-lastname-homework-xx, where xx is the HW number. So for example, I will name my folder as Gian-Alix-homework-01 for my first homework submission.
  2. Navigate to the /eclipse-workspace directory in your system, then you will be able to find the Java files that your wish to submit in one of the project folders.
  3. Copy-paste all files that you would like to submit to the firstname-lastname-homework-xx folder.
  4. Remember that each question should be in its own file.
  5. Once all files that you would like to submit are now in the folder, compress it using any of the zipping softwares. For Windows such as: WinRar, 7zip, or WinZip (no need to install if your machine already has one). For Mac, a compression utility software is already built-in so no need to install (simply right click on the folder, then click on Compress).
  6. Upload the zipped folder to the OSSFiles directory specified on the Classkick Homework; remember that each Homework gets a different link!
  7. Feel free to upload as many times as you wish. Previously uploaded files are overwritten. Only the last submission will be graded.

Feel free to ask me questions about homework submission (especially if you're not sure how or where to find the /eclipse-workspace directory) and I will be able to help you.

More Information About the Homeworks

There are three status for Homeworks, which you should see on the first page of any homework posted on Classkick:
  • Homework open - depicted by a yellow background, which means that the Homework is available to be edited/submitted to and the deadline has not yet passed.
  • Grading in process - depicted by an orange background, which means that the due date has already passed and your teacher is currently grading it; currently not editable
  • Marked - depicted by an green background, which means that your teacher has already graded it. Your teacher will make it editable again if in case you need to fix your corrections.
For each question of the Homework, it is either a 0/1. Then you will be graded with a Pass (50% or above) or Incomplete (below 50%) or No HW for not handing in the HW. Note that you also need to follow the instructions carefully to get a Pass. Your teacher will be providing feedback on Classkick.
