Course Description

This course is designed as an introductory programming course specifically in the programming language Java. In this course, students will learn basic programming concepts, such as variables, operators, conditional statements, writing iterative programs, simple data structures (e.g., arrays, lists, sets), recursion, basic object-oriented programming, and software testing. By the end of the course, students will gain programming skills needed in various courses such as AP-Computer Science, Computing Contest Courses, and ICS 3U. No prior programming experience needed.

Also, please try to do an advanced reading on the upcoming material for Monday (Procedural Programming); as it will be the last topic that will be covered to be included in the midterm on Thursday. There is one question from the homework about checking if an integer is odd. If we want to check if an integer is odd, the following code is wrong: 
    public static void main(String[] args) {
    // Assume n is the integer; you could try for positive n integers (e.g., 7, 15, 23, etc.)
    // which gives a correct answer but negative integers don't work (e.g., -7, -15, -23, etc.)
    if (n % 2 == 1) {
        System.out.println("The integer is odd.");
    }
}
    Here's a workaround. Option 1: check n % 2 != 0 in your if statement. Option 2: What you check instead is absN, where absN = Math.abs(n). 
    public static void main(String[] args) {
    // Option 1
    if (n % 2 != 0) {
        System.out.println("The integer is odd.");
    }

    // Option 2
    int absN = Math.abs(n);
    if (absN % 2 == 1) {
        System.out.println("The integer is odd.");
    }
}
