Course Description

This course is designed as an introductory programming course specifically in the programming language Python. In this course, students will learn basic programming concepts, such as variables, operators, conditional statements, writing iterative programs, simple data structures (e.g., tuples, lists, sets, dictionaries), recursion, basic object-oriented programming, and software testing. By the end of the course, students will gain programming skills needed in various courses such as Computing Contest Courses and ICS 3U. No prior programming experience needed.

Instructions for submission:

  1. First create a folder with the following filename: firstname-lastname-HWx, where x is the HW number. So for example, I will name my folder as Gian-Alix-HW1 for my first homework submission.
  2. Place all files that you would like to submit to this folder (i.e., copy-paste). For each file of your homework submission, you will submit either one of the following file types:
    • A .py file - this will be the most common file type that you will be submitting. Navigate to the /PycharmProjects directory in your system, then you will be able to find the file that your wish to submit in one of the project folders.
    • A .pdf file - there are some homework exercises that does not necessarily require a code solution, but perhaps a math solution or an explanation from you. In such cases, you can create a document (i.e., Microsoft Word, Google Doc, etc.), type your answers there, and then export it to PDF format.
    • A .txt file - I will be specifying if a simple txt file is needed for a specific homework submission
  3. Once all files that you would like to submit are now in the folder, compress it using any of the zipping softwares. For Windows such as: WinRar, 7zip, or WinZip (no need to install if your machine already has one). For Mac, a compression utility software is already built-in so no need to install (simply right click on the folder, then click on Compress).
  4. Upload the zipped folder to the following directory. Access it here. Feel free to upload as many times as you wish. Previously uploaded files are overwritten. Only the last submission will be graded.

Feel free to ask me questions about homework submission (especially if you're not sure how or where to find the /PycharmProjects directory) and I will be able to help you.
