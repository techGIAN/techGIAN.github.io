Deep Reinforcement Learning of Spatial Trajectory Pathlet Dictionaries (Master's Thesis)

The construction of trajectory pathlet dictionaries using Deep Reinforcement Learning

  • Collected, preprocessed and map-matched large ~3.8 million raw GPS vehicular mobility trace datasets with the help of OpenStreetMaps and the Open Source Route Machines (OSRM)
  • Constructed small sets of building blocks representing wide ranges of spatial trajectories, called trajectory pathlet dictionary
  • Proposed & implemented a Reinforcement Learning method to learn dictionaries based on maximal utility theory
  • Utilized Tensorflow's tf-agents package to implement deep reinforcement learning models.
  • Empirically showed that the model can outperform the SotA by learning 66% smaller dictionaries than the baselines, can reconstruct 85% of evaluation trajectories given only half of the dictionary, and can converge as early as 15 iterations of running the model
  • Click here for the Github Repository

Spatiotemporal Trajectory Similarity Learning (Master's Thesis)

The retrieval of the list of the (ranked) top $k$ trajectories spatially and temporally similar to a query

  • Proposed a Representation method for learning spatial-temporal features of trajectories for similarity learning
  • Utilized PyTorch package mainly to implement the trajectory representation learning model.
  • Empirically showed that the model can outperform the SotA baselines by up to 38%. The model is characterized by its accuracy, versatility, generalizability, robustness, speed, and scalability
  • Click here for the Github Repository

Epidemic Spreading with Trajectory Data Mining (COVID-19 Research Project)

Examining the spatial-temporal dynamics of epidemic spread using mobility data.

  • Created an infection risk map for ~259 million Points of Interests (POI) data extracted from SafeGraph that is based on mobility data (i.e., simulated trajectories) of over ~3,000 individuals through a mobility simulator called SUMO
  • Proposed personalized trip recommendation systems (through a web platform) that aim to minimize travel time and distance while optimizing infection risk, with the goal of supporting individual decision-making and informing public policy.
  • Visualized POI visits data through static and animated heatmaps and temporal patterns through $K$-means clustering
  • Used Pandas for data preprocessing, GeoPandas for working with spatial data, Scikitlearn for clustering, Matplotlib and Seaborn for visualizations, and Folium for generating heatmaps
  • Click here for the Github Repository.
